携帯番号でのアポ取得 対応ガイド

今までやっていた「Zoom にログイン → 録音を探す → ダウンロード → アポ取得報告に差し替え」の作業はもう不要です。
以下の手順で進めれば、録音の紐づけ・AI による報告書自動生成まで Spanavi 内で完結します。

1こちらから社長の携帯にかけた場合

架電する前

  1. その企業の1画面集中ページを開く
  2. 右側の キーマン携帯 欄に携帯番号を入力
  3. 入力欄の外を一度クリック → 自動保存されます

電話をかける

  1. キーマン携帯 の隣の 発信 ボタンを押す
  2. Zoom Phone から発信 → 通話開始（録音は自動）
注意： 必ず Spanavi の「発信」ボタンから掛けてください。個人スマホや会社の電話アプリから直接かけると、録音が Spanavi に残りません。

通話後・アポが取れたら

  1. 普段通り アポ獲得 ステータスを押す
  2. アポ取得報告画面が開く（録音 URL はセット済み）
  3. 文字起こし＋AI添削 ボタンを押す → 温度感・面談経験・将来検討・その他が自動で入る
  4. 内容を確認して 保存

2社長の携帯から着信を取った場合

着信を取る

  1. Zoom Phone（PC または モバイルアプリ）で着信を取る
  2. 通話 → アポ獲得まで持っていく → 電話を切る
注意： 個人スマホで取る場合は、必ずZoom Phone のモバイルアプリで取ること。通常の電話アプリで取ると Spanavi には何も残りません。

着信対応画面で処理

  1. 左メニューの 着信対応 をクリック
  2. 該当の着信行を見て、下の AかB のどちらか
A：企業名が表示されている
そのまま アポ取得 ボタンを押す
→ 報告画面が企業情報＋録音 URL 入りで開く
B：「企業に紐づける」が出ている
ボタンを押す → 検索窓に企業名を入力 → 候補をクリックして紐づけ
→ 行に アポ取得 が出る
※ 紐づけた携帯番号は次回以降の着信で自動認識

アポ取得報告

  1. 文字起こし＋AI添削 ボタンを押す
  2. 温度感・面談経験・将来検討・その他が自動で入る
  3. 内容を確認して 保存

3携帯番号を登録せずにアポ報告を書いてしまった場合

慌てなくて大丈夫。後からでも救済できます。

  1. その企業の1画面集中ページに戻る
  2. キーマン携帯 欄に番号を入れて保存（外をクリック）
  3. アポ一覧 で該当のアポを開く
  4. 文字起こし＋AI添削 ボタンを押す
  5. 過去の録音が後から引っ張られて、報告書が AI で再生成される

4まとめ表

パターン 操作する場所 主なアクション
① こちらから携帯にかける 1画面集中ページ 「キーマン携帯」入力 → 「発信」ボタン → アポ獲得 → AI添削
② 携帯から着信を取る 着信対応タブ 「アポ取得」ボタン（必要なら企業紐づけ）→ AI添削
③ 後から気づいた 1画面集中ページ＋アポ一覧 「キーマン携帯」入力 → アポを開いて「文字起こし＋AI添削」
おまけ：キーマンが代わった 1画面集中ページ 「キーマン携帯」欄を空にする（古い番号での紐づけも自動解除）
おまけ：支店・営業所番号 1画面集中ページ 「別事業所番号」欄に入れる（同じ仕組みで自動紐づけ）
