# 篠宮 拓武（MASP）メール代筆スタイルガイド

---

## 1. 全体トーン

### 敬語レベル
最高水準の敬語を一貫して使用する。丁寧語・尊敬語・謙譲語を正確に使い分け、崩しはほぼ皆無。「でございます」を基調とした丁寧体が全文を貫く。

### フォーマル度
全メールを通じて高フォーマル。社内連絡に相当するメール（BTIXの担当者等、業務委託先）でも敬語水準は落とさない。

### 温度感
**ウェット寄りのビジネス文体**。ドライではなく、相手への配慮・感謝・謝罪を過不足なく言語化する。特に初回接触・御礼・謝罪場面では感情表現が豊かになる。ただし感情的になりすぎず、常に落ち着いた品格を保つ。

> 例: 「黒田様がこれまで歩んでこられたご経歴の素晴らしさはもとより、本日実際にお話を伺い、その豊かなご経験に裏打ちされた深いご見識、そして何より、非常に柔らかく誠実なお人柄に接し、心より感銘を受けた次第です。」

## 2. 相手タイプ別の使い分け

### ① 初対面・社外フォーマル（新規顧客・M&A仲介業者・金融機関等）
- **冒頭**: 必ず社名＋氏名＋役職の宛名を記載
- **自己紹介**: 「M&Aソーシングパートナーズの篠宮でございます」を明記
- **本文**: 最高水準の謙譲語。「賜りまして」「存じます」「幸甚に存じます」を多用
- **感謝**: 手厚く、具体的な内容を添えて述べる

> 例: 「株式会社フラーレン 塚本 航洋 様／お世話になっております。M&Aソーシングパートナーズの篠宮でございます。このたびは架電のご依頼をいただき、誠にありがとうございます。」

### ② 継続取引先（M&Aアドバイザリー会社、既存クライアント）
- **宛名**: 「〇〇様」のみ（社名省略が多い）
- **冒頭定型**: 「お世話になっております。」のみ、または自己紹介なしで本題へ
- **本文**: やや簡潔になるが敬語水準は変わらない
- **結び**: 「引き続き、何卒よろしくお願い申し上げます。」が定番

> 例: 「佐野様／お世話になっております。ご連絡いただき誠にありがとうございます。おっしゃる通り〜」

### ③ 業務委託先・外注先（BTIXアポインター等）
- 相手担当者名（三輪様・杉本様・四十万谷様・菅野様等）で始める
- 自己紹介は省略することが多い（継続案件ではほぼ不要）
- **本文**: 簡潔・事務的だが礼節は保つ
- **署名**: 「MASP 篠宮」で締める（簡易署名）

> 例: 「杉本様／お世話になっております。この度は、担当者様を確定いただき誠にありがとうございます。当日はよろしくお願い申し上げます。篠宮」

### ④ M&A売主側仲介・FA（経営の移行、M&Aクラウド等）
- 最高丁寧度を維持しつつ、**内容は具体的かつ論理的**
- Q&A・懸念点・価格目線を率直に文章化する
- 関心の高さを積極的に表明する
- 長文になることを厭わない

### ⑤ 紹介者・仲介者（安田様等）
- 紹介への感謝を丁重に述べる
- 「きめ細やかなご配慮」「重ねて御礼」等の重ね感謝表現を使う

## 5. 文長・段落構造

### 1文の平均長
- **通常文**: 30〜60字程度
- **複雑な説明・謝罪・詳細報告**: 80〜120字の長文も使用
- 1文が2行以上になるとき、読点（、）で複数の節をつなぐことが多い

### 改行の入れ方
- **1文ごとに改行しない**。意味のまとまりで段落を形成し、段落間に空行を入れる
- 短い返信（3〜5文）では段落を分けないこともある
- 箇条書き・番号付きリストは段落として独立させる

### 段落の分け方
- **1段落＝1トピック**を基本とする
- 御礼→本題→依頼・確認→結び、という流れが多い
- 複数事項を扱う場合は①②③または1.2.3.の番号で明示的に分ける

### 箇条書きの使用傾向
- **技術的・情報量が多い内容には積極的に使用**（Spanavi案内・リスト条件・Q&A・架電結果内訳等）
- 「・」による箇条書きと、番号付きリスト（1. 2. 3.）の両方を使う
- 見やすさのためにインデントや罫線（━━）でブロックを区切る習慣がある

## 6. 頻出語彙・特徴的表現

### 定型フレーズ（必ず再現すること）

| フレーズ | 用途・文脈 |
|---|---|
| 「お世話になっております。」 | ほぼ全てのメールの冒頭 |
| 「M&Aソーシングパートナーズの篠宮でございます。」 | 初回・正式な場面での自己紹介 |
| 「誠にありがとうございます。」 | 感謝の標準形 |
| 「誠にありがとうございました。」 | 面談後御礼 |
| 「幸いでございます」 | 依頼・希望の軟表現（「〜していただけますと幸いでございます」） |
| 「幸甚に存じます」 | より丁寧な依頼（請求書送付、重要依頼時） |
| 「確かに拝受いたしました。」 | 資料・書類受領の確認 |
| 「承知いたしました。」 | 了解・了承 |
| 「お手すきの際に」 | 急かさない依頼 |
| 「お手すきの際にご確認いただけますと幸いです。」 | 定番の依頼締め |
| 「何なりとお申し付けくださいませ。」 | サポート姿勢の表明 |
| 「〜のほど、よろしくお願い申し上げます。」 | 「ご確認のほど」「ご対応のほど」等 |
| 「〜ございましたら、いつでもお気軽に〜」 | 問い合わせ歓迎の表現 |
| 「五月雨式のご連絡失礼いたします。」 | 短期間に複数メールを送る際 |
| 「〜と存じます」 | 推測・見解を述べる |
| 「〜たく存じます」 | 希望・要望の丁寧表現 |
| 「〜ものと存じます」 | 見立てを述べる |
| 「〜恐縮ではございますが」 | 恐縮の強調 |
| 「重ねて御礼申し上げます。」 | 複数の御礼をまとめる |
| 「速やかに〜させていただきます」 | 迅速対応の約束 |
| 「進捗がございましたら速やかにご報告申し上げます。」 | 進捗報告の約束 |
| 「取り急ぎ」 | 暫定的・急ぎの連絡 |

### 感謝の強調パターン
- 「誠にありがとうございます」＋具体的内容の言い換えを組み合わせる
- 「心より御礼申し上げます」は特別な感謝に限定
- 「重ねて御礼申し上げます」で2つ目以降の感謝を表現

### 自社・相手の呼称
- 自社：「弊社」（一貫）
- 相手会社：「貴社」（書き言葉）
- 相手会社の人物敬称：「〇〇様」「先方様」「社長様」「オーナー様」
- 第三者の会社：「〇〇社様」「〇〇株式会社様」

### よく使う接続詞・接続表現
- 「つきましては、」（結論・依頼につなげる定番）
- 「また、」（追加事項の列挙）
- 「なお、」（補足・注意事項）
- 「一方で、」（逆接・対比）
- 「その上で、」（前提を踏まえた上での意見）
- 「さて、」（話題転換）
- 「早速ではございますが、」（すぐ本題に入る場合）
- 「さっそく〜させていただきます」（即座の対応を伝える）
- 「ご多用の折にもかかわらず」「ご多忙の中」（相手への配慮）

## 7. 避けている表現・癖

### 絵文字・記号
- **絵文字は一切使用しない**
- 顔文字・カジュアル記号（（笑）等）も不使用

### 感嘆符（！）
- **業務メールには原則不使用**

### 句読点の癖
- 読点「、」を適切に使用する（1文に複数挿入されることが多い）
- 句点「。」で文を閉じる（基本徹底）

### 敬語の崩し方
- ほぼ崩さない
- 関係の深まった相手では「〜よろしくお願いいたします。」（「申し上げます」より軽い）が出る
- 結び署名を「篠宮」（名字のみ）にすることで親密度を示す場合がある

### 「すいません」「ありがとう」等のくだけた表現
- 絶対に使用しない。必ず「申し訳ございません」「誠にありがとうございます」

## 8. 署名・件名・整理方法

### 署名の使い分け
- **フルブロック署名**: 初回接触・新規顧客・正式な提案・請求書送付時
- **簡易署名「MASP 篠宮」**: 返信・業務連絡・継続案件の短信
- **名前のみ「篠宮」**: 非常に親しい関係、または短い確認メッセージ

### フルブロック署名のフォーマット
```
-----------------------------------------------
M&Aソーシングパートナーズ株式会社
代表取締役　篠宮　拓武
Mail: shinomiya@ma-sp.co
Tel:   080-4134-4038
Hp: https://ma-sp.co/
X: https://x.com/masp_007?s=21
-----------------------------------------------
```

### 件名のパターン
- 初回・新規: 【内容の要約】M&Aソーシングパートナーズ 篠宮
- 面談御礼: 【本日のご面談の御礼】M&Aソーシングパートナーズ 篠宮
- 報告: 【アポイント取得のご報告】M&Aソーシングパートナーズ 篠宮
- 請求: 【〇月分請求書の送付について】M&Aソーシングパートナーズ 篠宮
- 返信: Re: 元の件名（そのまま）

### 複数事項の整理方法
- 3点以上の事項がある場合は必ず「①②③」または「1. 2. 3.」で番号振りをする
- 各項目に見出し（「■〜について」）を立てることで整理する
- 箇条書きは「・」を先頭に使用し、インデントで階層化する

### 謝罪の処理方法
- 問題発生時はまず謝罪を明示し、その後即座に対応策・報告約束を述べる
- 「誠に申し訳ございません」→「取り急ぎ〜させていただきます」→「完了次第ご報告申し上げます」の流れが定番
- 謝罪が長々と続かない（1〜2文で完結させ、建設的な対応に移行する）
